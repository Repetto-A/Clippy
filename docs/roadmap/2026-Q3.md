# Roadmap — Clippy 2026 Q3

> Backlog que alimenta el loop de `../autonomy-charter.md`. El agente toma el **próximo rep no
> bloqueado de mayor prioridad**, lo shippea con su done-when, y actualiza este archivo.
> Cada rep es la versión más chica que aporta valor (ship-learn-next).

**Leyenda de estado:** `todo` · `wip` · `done` · `blocked (motivo)`

---

## Bloqueante fundacional

- **R0 — Etiquetar una clase (baseline real).** `blocked (requiere Ale)`
  Sin golden set real, los cambios de motor no tienen vara (ADR-0003). Es lo único que el
  agente no puede hacer solo. **Done-when:** una clase con `labels.json` completo + `eval`
  corrido → `eval_report.json` como baseline vigente.

---

## Prioridad 1 — Confianza y transparencia (sin depender de R0)

- **R1 — Timeline semántico (ADR-0006).** `done`
  Endpoint de señales agregadas + panel heatmap navegable en el editor.
  **Done-when:** ver ADR-0006. No cambia comportamiento del motor.

- **R2 — Explicabilidad del clip.** `done`
  Mostrar en el editor *por qué* se eligió cada clip: sub-scores de rúbrica + snippet del hook
  + razón. Reusa datos existentes. **Done-when:** cada card de clip muestra rúbrica desglosada
  y el texto del hook; sin cambios de datos.

- **R3 — Pulido de export.** `done`
  Preview con selector de formato, nombres de descarga legibles, badge de render en cards.
  **Done-when:** preview fiel + descarga con nombres legibles por formato.

## Prioridad 2 — Aprendizaje del sistema (mejor con R0)

- **R4 — Import de performance externo (ADR-0005), fase manual.** `done`
  `performance.json` + import CSV/JSON mapeado a `clip_id` + reporte de correlación.
  **Done-when:** ver ADR-0005 (fase manual, sin fetcher automático).

- **R5 — Sugerencia de pesos de rúbrica.** `blocked (R0, R4)`
  Correlacionar sub-scores vs performance y sugerir `rubric_weights`. Aplicación detrás de flag
  + validación vs baseline. **Done-when:** reporte de sugerencia; nunca auto-aplica sin OK.

## Prioridad 3 — Escala de formatos (cuando haya 2º caso real)

- **R6 — Formalizar perfil `training` (ADR-0007).** `done`
  Introducir el concepto `profile` con `training` como default, sin cambio de comportamiento.
  **Done-when:** job creable con `profile=training`; defaults de prefs derivan del perfil.

- **R7 — Perfil `podcast`.** `blocked (requiere pedido de Ale)`
  Reframe por speaker-switch, rúbrica y caption de podcast. **Done-when:** un podcast de prueba
  produce clips razonables con el preset.

## Continuo — Salud del proyecto

- **RC1 — Robustez de videos largos.** `wip`
  Completar validación end-to-end de la clase de 93 min (chunking ASR + resume).
  **Done-when:** propose→render completo sobre la clase real sin intervención.
- **RC2 — Cobertura de tests + lint limpio.** `done` (73 tests verdes tras R3)
- **RC3 — CHANGELOG + handoff notes.** `todo` — actualizar por cada rep shippeado.
